True headless capture

Android Foreground Service + MediaCodec H.264 High, 1080p60, CBR ~7 Mbps, GOP=1 s (IDR/sec). Writes 1 s CMAF m4s into a 10:00 rolling buffer.

Director-led HQ review

Director pulls only the last 30 s from selected cameras over HTTP (LAN) and plays progressively via AVFoundation. No preview-quality stream.
